# AEON — AI Memory Sync (Waitlist)

AEON is a simple, mobile-first platform designed to solve one core problem:

👉 Repeating context across multiple AI tools.

Today, users switch between tools like ChatGPT, Claude, Gemini, and others — but every time they switch, they have to start from scratch.

AON aims to eliminate this friction.

---

## 🚀 What This Project Is

This repository contains the **waitlist landing page** for AON.

The purpose of this website is to:

* Introduce the idea of AEON
* Explain the problem and solution
* Collect early user interest (waitlist signups)

---

## 💡 The Idea

AEON will allow users to:

* Connect multiple AI tools in one place
* Sync their context/memory across platforms
* Avoid repeating the same information again and again

**In simple terms:**

> "Use any AI tool without losing your context."
